## 简介
这是一个饥荒（DST）mod开发demo，内置很多从工坊下载的mod，目的是通过这些参考资料学习饥荒mod的开发

## 目录结构
```bash

DIR:dst-mod-lean                                            # 
   |-- MyMods/                                              # 「demo」 正在编写的mods
   |-- dst-scripts-public/                                  # 「dst-scripts-public」
   |-- test/                                                # 「test」 实时测试文件夹
   |-- mode_refs/                                           # 「mode_refs」 从工坊下载的参考模组
   |-- tools/                                               # 「tools」
   |-- 开发资料/                                             # 「tools」
   |-- 破解/                                                # 「破解」 一些机密混淆了的mod
   |-- README.md                                            # 
   |-- modmain.lua                                          # 
   |-- modinfo.lua                                          # 
   `-- DoNotStarveTogether.lnk                              # 

```
